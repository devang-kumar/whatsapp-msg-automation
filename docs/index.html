Setup instructions
  1. Click Download for Windows above
  2. Double-click WhatsApp Bulk Messenger.exe
  3. If Windows SmartScreen appears, click More info → Run anyway
  4. Your browser opens automatically at localhost:5000
  5. Scan the WhatsApp QR code once — you're done

⚠️ Google Chrome must be installed on your PC.

  1. Click Download Source (.zip) above and extract it

  3. Double-click start.bat inside the extracted folder
  4. Dependencies install automatically on first run
  5. Browser opens at localhost:5000
start.bat
  1. Click Download Source (.zip) above and extract it
  2. Make sure Python 3.10+ is installed:
    macOS: brew install python  ·  Ubuntu: sudo apt install python3 python3-venv
  3. Open Terminal inside the extracted folder and run:
chmod +x start.sh ./start.sh

Browser opens automatically at localhost:5000. Google Chrome must be installed.
